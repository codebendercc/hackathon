Hey you PHP developers
developers, developers, developers, developers

Βαρέθηκες τα pitching event και τα "hackathons" όπου η ιδέα ή το πλασάρισμα, μετράει περισσότερο από την υλοποίηση;

Θέλεις κάτι hardcore όπου θα μπορέσεις επιτέλους να αναμετρήσεις τις... γνώσεις σου με άλλους developers;

Αν ναι σε περιμένουμε στο 1ο All-night Web Development Hackathon by codebender.

Δημιούργησε ένα web app, service ή API, με όποια ιδέα έχεις, ακολουθώντας τα industry best practices.

Πότε θα γίνουν όλα αυτά;

Το hackathon θα ξεκινήσει ταυτόχρονα και σε Αθήνα και Πάτρα το Σάββατο 4/7/15 στις 09:00 και θα τελειώσει στις 15.00 της επόμενης μέρας (Κυριακή). Αν δεν ξυπνήσεις νωρίς μπορείς να έρθεις αργότερα αλλά θα έχεις λιγότερο χρόνο στην διάθεσή σου.

Ψήθηκες;

Η καλύτερη υλοποίηση μεταξύ των δύο πόλεων κερδίζει 700€ και η επόμενη καλύτερη στην άλλη πόλη 500€.

Spend them wisely young padawan.

Ο κώδικας μετράει

Πως θα γίνει η επιλογή

Κάποια ενδεικτικά σημεία που θα βαθμολογηθούν με σειρά προτεραιότητας

  • Ποιότητα κώδικα (με αυτοματοποιημένα εργαλεία όπως το phpmd)
  • Κάλυψη unit/functional test (PHPUnit)
  • Χρήση VCS, κατά προτίμηση git
  • Εύρος δυνατοτήτων και ευχρηστία του API
  • Security principles
  • Ύπαρξη frontend
  • Πρωτοτυπία/Χρησιμότητα/Fun factor
Αναλυτικές πληροφορίες για τον τρόπο βαθμολόγισης θα δωθούν στην έναρξη του hackathon

Απαραίτητες προϋποθέσεις

  • Aτομική συμμετοχή και όχι σε ομάδες
  • Γλώσσα προγραμματισμού PHP για το backend
  • Χρήση κάποιου open source web framework και γενικότερα open source βιβλιοθηκών/κώδικα
  • Ο κώδικάς σας να δημοσιευτεί με open source license σε public repo (πχ github)
  • Να περιλαμβάνεται readme ώστε να μπορεί να ελεγχθεί με απλές οδηγίες (δεν δεχόμαστε "it works on my laptop")
  • Να φέρετε τον δικό σας υπολογιστή (dah!)
